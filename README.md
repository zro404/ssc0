# Tiny Selfhosted Scheme Compiler
Compiles Scheme programs (R<sup>5</sup>RS subset) to x86\_64 linux executables

> WORK IN PROGRESS

### Dependencies:
1. [MIT/GNU Scheme](https://www.gnu.org/software/mit-scheme/)
2. [Netwide Assembler (NASM)](https://www.nasm.us/)

### References:
1. [R<sup>5</sup>RS Standard](https://standards.scheme.org/official/r5rs.pdf)
